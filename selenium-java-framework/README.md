# Advanced Selenium Java Framework

This framework demonstrates a scalable Selenium automation design using Java, Maven, TestNG, and Extent Reports.

## Features

- Page Object Model
- Config file support
- Reusable driver factory
- Screenshot capture on failure
- Extent Report integration
- TestNG listener implementation
- Login, cart, and checkout automation

## Application Under Test

- https://www.saucedemo.com/

## Run tests

```bash
mvn clean test
```

## Reports

Extent report:
```text
test-output/ExtentReport.html
```

Screenshots on failure:
