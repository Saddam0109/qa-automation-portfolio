# API Testing Framework

This framework demonstrates API automation using Python, Requests, and Pytest.

## Features

- Reusable API client
- Config-based base URL
- JSON test data
- HTML reporting
- GET and POST sample validations

## Installation

```bash
pip install -r requirements.txt
```

## Run tests

```bash
pytest
```

## Report

After execution, open:

```text
reports/api-report.html
```
